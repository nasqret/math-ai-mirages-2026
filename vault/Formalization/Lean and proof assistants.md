# Lean and proof assistants

Formal proof provides a machine-checkable endpoint, but the statement,
definitions, axioms, imports, and trust boundary remain human governance
choices. The lecture will use the EML project to show the complete loop:

informal paper → decomposition → theorem specifications → proof search → kernel
check → audit → reusable library.

The kernel certifies a term against a formal statement. It does not certify that
the formal statement matches the intended theorem or that the theorem is novel.
