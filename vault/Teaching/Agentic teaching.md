# Agentic teaching

Public deployed examples:

- cryptography: 45 chapters;
- neural networks: 42 chapters, 329 slides, 22 applets;
- 25-hour Codex course for university staff.

The strong outcome is not answer generation. It is a coherent course ecosystem:
notes, exact code, interactive artifacts, tests, and a repository another person
can inspect.
