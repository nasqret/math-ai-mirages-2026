# Agentic teaching

Public deployed examples:

- cryptanalysis: 45 chapters, 15 parts, nine applets; direct artifacts include
  Rejewski's attack and lattice explorers;
- neural networks: 42 chapters, 329 slides, 22 core applets; direct artifacts
  include an interactive UAT proof and tokenizer comparison;
- 25-hour Codex course for university staff: local repositories, plan, memory,
  journal, validation, Git, and deployment.

Concrete student-facing specifications include reverse-mode autodiff from
scratch with PyTorch checks, a 24-paper reproduction catalogue, Tiny GPT, and
Polish character-RNN projects.

The strong outcome is not answer generation. It is a coherent course ecosystem:
notes, exact code, interactive artifacts, tests, and a repository another person
can inspect.
